<details>
<summary><b>🔸 Deploy on VPS / Localhost</b></summary>

1. Fork & Star ⭐
Click Fork (top-right of GitHub repo)

Then click Star ⭐ to support this project!

2. Get Your Fork URL
https://github.com/<your-username>/NomadeHelpBot.git
3. Setup Your VPS
Bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y git python3 python3-pip python3-venv tmux nano
4. Clone Your Fork
Bash
git clone https://github.com/<your-username>/NomadeHelpBot.git
cd Nomade
python3 -m venv venv
source venv/bin/activate
5. Install Dependencies
Bash
pip install --upgrade pip && pip install -r requirements.txt
6. Configure the Bot
Bash
nano .env
⚙️ Required Fields

Ini, TOML
# Telegram API
API_ID=
API_HASH=
BOT_TOKEN=

# MongoDB
MONGO_URI=
DB_NAME=Cluster0

# Owner and Bot Info
OWNER_ID=
BOT_USERNAME=PikachuuHelpBot

# Links & Visuals
SUPPORT_GROUP=https://t.me/+UhZo8ZsUECYyYWI1
UPDATE_CHANNEL=https://t.me/pikachuu_updates
START_IMAGE=https://files.catbox.moe/9qftmz.jpg
✅ Save with: Ctrl + O, then Enter

❌ Exit with: Ctrl + X

7. Run the Bot
Bash
tmux new -s groupbot
source venv/bin/activate
python3 main.py
➡️ Detach: Ctrl + B, then D

</details>
